The strategy one line

Sell parlays, never buy them. We quote only the NO side of a combo (yes_bid=0), so we can only ever land long NO and win when a parlay busts — which it does ~93% of the time. Markup is deliberately thin (~1¢): settlement data proved wider markup is adversely selected, not safer.

Pricing · the moat

Correlation, by sport.

The joint layer is the only thing modeled bottom-up. Each sport is its own calibration with its own out-of-sample gate — a model that doesn't beat the copula on held-out data doesn't ship. Pick a sport.

The correlation surface is far from complete. Soccer is strong and nearly done (final tests pending) — it only hardens from here. MLB is the ACTIVE workstream (finalization in process since 2026-07-09: classify all prop families → measure every allowed pair → verify → backtest). NBA gated, UFC & Tennis unbuilt. This is the biggest open work in the whole system.

Risk · the edge

The book, the caps, the last look.

Pricing is a commodity; discipline is the edge. Risk gates every quote before it leaves and again at confirm, and flattens the book on a kill.

Coded, not proven. Every module below is written and unit-tested, but none of it has run against live flow — the exposure book, mass-acceptance worst case, and Monte-Carlo engine have never been stress-tested in reality. "coded" here means the code exists and the tests pass, not that we've seen it hold up.
Self-grading defense. Markouts are computed against BOTH our fair and raw leg mids (a biased fair can't grade itself), and the EV ledger is scored on aggregate expected-edge vs realized P&L — never single outcomes. Settlements are the one ruler the model can't bend.
