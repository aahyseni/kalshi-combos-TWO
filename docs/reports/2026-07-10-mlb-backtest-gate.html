Method

gather: MLB-strict combos + marginal snapshots from the recorder DB (read-only, bounded chunks; 16.05M rfq rows scanned) · clearings from Kalshi's authed trade tape, complete-per-combo through ~2026-07-11 01:35Z (poller-gap-immune) · per-leg settlement/expiry from the public API · strictly-pregame filter keeps prints before min-over-legs(expected_expiration − 4h for KXMLB). price: blind driver over the LIVE pricing modules + shipped 165-entry config (rung-key chain: exact rung → un-runged oriented → plain, no interpolation); the legacy config is a runtime deep-copy override (engine untouched, CLAUDE.md rule 8); fairs computed only for tape-printed combos via a prices-free ticker list (selection cannot bias a fair). per-print mode: fair recomputed at each print's own timestamp (53,022 prints; 5,782 unpriced for missing pre-print snapshots). analyze: this page. Reports channel: docs/reports/.
