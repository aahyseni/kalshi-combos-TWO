### C.2 The solver (the ONE copula, inverted)

- **Rank-1**: empirical joint frequency P(A), P(B), P(A∩B) over N historical games.
- **Rank-2**: invert THE SAME Gaussian copula the pricer runs (`gaussian_copula_joint_prob`) via bisection (`implied_rho()`) to turn P(A∩B) into a copula ρ that is a drop-in `pair_rho`; report 99% CI (binomial SE pushed through the solver). Tetrachoric ρ estimator, BVN validated to 1.1e-16 / 2e-16 vs scipy.
- **Rank-3 (gold)**: conditional-MLE ρ fit on each game's OWN devigged closing-line marginals (Owen's-T BVN), then OOS-gated.
- **Rank-4 (soccer only)**: Dixon-Coles structural scoreline model (independent Poisson 90' + low-score τ; ET at et_factor; player goals = multinomial thinning). NFL/NBA/WNBA get a margin-total bivariate normal; MLB gets a NegBin runs grid.
- **Rank-5 (Ising / MC)**: evaluated (results_ising.md) — deferred; copula is v1.
- Marginals are ALWAYS taken live from Kalshi leg books at quote time. History calibrates ONLY the joint layer. Rationale: the launch-angle era moved the MARGINALS (P(HR) 0.089→0.107) not the copula — exactly why the architecture calibrates only the joint and takes marginals live.

### C.4 Orientation / rung / frame / window conventions (the pricing grammar)

- **Orientation** `:same`/`:opp`/`:tie`/`:team`: `:same` = prop player's team IS the ML/spread YES team; for batter-stat×player_ks `:opp` IS the facing case. Both sides measured directly — **never negate :same to get :opp** (asymmetric, gap grows with line).
- **Rung** `:rN` = Kalshi ticker line integer (leg settles YES iff stat/margin > N−0.5; props N+, spread wins-by-N+). Rung-keyed families: player_hit/hr/tb/hrr/spread. player_ks/total/moneyline/rfi NEVER carry rungs (self-median/season-median frames). When both legs are rung-keyed, suffixes CHAIN in pair_key leg order. **No interpolation/extrapolation** — exact rung key → un-runged oriented → plain → fail-closed default.
- **Frame**: batter-prop cells = all-PA batter-game frame EXCEPT player_hrr|total = STARTERS frame; frame convention centralized in ONE place (`shape_in_leg_frame` negates ρ into the leg frame — the L1 lesson).
- **Window / containment**: A⟹B exact windows priced as P(B)−P(A) (band arithmetic, no ρ); nested bands (yes-low + NO-high) = P(low)−P(high) exact via super-leg collapse; conditional super-legs isolated from same-game companions (fail-closed).
- **Farmable**: True ONLY on airtight ONE-official-record logical tautologies (5 soccer, 0 MLB — the 48h rain-scalar breaks airtightness; S8 cross-scope spans two records = unverified lemma = farmable=False).

### C.5 The wire-list format convention (the measurement→config artifact)
